Memory

Essentially, memory is how a computer remembers. Memory can be divided into 3 catagories: ROM, RAM, and Long Term. Memory is used to store information the computer needs to use later, like programs, startup sequinces, images, or text documents.

Examples:


  • ROM, or Read Only Memory, usualy contains simple algorithms, startup sequinces, the like; things that the user cannot change.
  • Ram, or Random Access Memory, is used by components to temporarally store information. Things like what the screen is displaying, unfinished algorithms, or basically any temporary data- for example, a word document is stored on the ram until you save it, transfering it into Long Term memory.
  • Long Term Memory is stored on Hard disks or Solid State Drives (SSDs). This stores programs the user can interact with- like video games, search engines, etc. Long term memory tends to be the slowest of the lot in terms of request time.
